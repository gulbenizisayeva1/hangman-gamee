# Hangman Game 

This project is a classic **Hangman Game** built using **HTML**, **CSS**, and **JavaScript**.  
The user tries to guess a hidden word by selecting letters. Each incorrect guess increases the hangman drawing. The game ends when the user either guesses the word correctly or reaches the maximum number of wrong guesses.


## Features
- Random word generation
- Letter selection keyboard
- Wrong guess counter
- Visual hangman progress
- Game reset functionality
- Responsive design
